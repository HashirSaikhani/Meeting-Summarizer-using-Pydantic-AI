Extracted Meeting Features:

- Pipeline design for client onboarding
- Support for various data formats (structured and unstructured)
- Backend for data parsing, conversion, and storage
- Structured data ingestion
- Unstructured data ingestion
- Dynamic fact updates via admin dashboard
- Automatic embedding regeneration for updated facts
- WhatsApp integration (live listening and historical data import)
- Filtering and cleaning of imported historical data
- Human escalation system with transparent handover
- Human agent replies via admin dashboard or WhatsApp Business
- Specialized escalation for specific use cases (e.g., payment verification)
- Escalation tracking dashboard with metrics
- Initial focus on text data, meeting transcripts, and audio-to-text
- Future support for image data
